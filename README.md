# RS Flutter Workspace

A Flutter development workspace for cross-platform mobile application development.

## Project Structure

- **lib/**: Main Dart source code
- **android/**: Android-specific configuration
- **ios/**: iOS-specific configuration  
- **web/**: Web platform configuration
- **test/**: Unit and widget tests
- **assets/**: Images, fonts, and other resources

## Getting Started

### Prerequisites
- Flutter SDK
- Dart SDK
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)

### Running the App

To run this Flutter application:

```bash
cd rs_flutter
flutter run
```

### Available Tasks

You can use VS Code tasks to run common commands:
- **Flutter: Run App** - Run the Flutter application
- **Flutter: Clean** - Clean the project build files
- **Flutter: Get Dependencies** - Fetch project dependencies
